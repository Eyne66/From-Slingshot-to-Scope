# 从弹弓到瞄准镜 / From Slingshot to Scope

一个关于学习、AI 工具、考研备考、认知误差与自我校准的中文交互思想实验。

它不是传统工具型开源项目，也不是单纯展示 AI Coding 技术的网页。它更像一个小型认知模型：把一次真实学习复盘里的感受，转译成可以被拖动、观看和追问的代码原型。

这个仓库重点记录的是：一个想法如何被整理、压缩、结构化，并最终变成一个可交互的网页。

## 项目缘起

这个项目来自一次真实的学习复盘。

在做考研英语真题时，我发现自己真正的提升并不是“知道更多答案”，而是开始意识到自己会如何被误导：被熟悉感骗，被假线索带偏，被先入为主牵着走，也会用工具制造一种“我好像懂了”的幻觉。

于是我把这种感觉转化为一个隐喻：

> 人的成长不是突然变强，而是从“弹弓期”到“拿到枪但仍然近视”，再到“拥有瞄准镜”的过程。

## 核心隐喻

### 1. 弹弓期

工具弱，目标也模糊。

人靠兴趣、冲动、零散尝试行动，不知道自己真正瞄准什么。

### 2. 枪但近视期

工具变强了。

AI、资料、真题、教程都在手里，但判断能力还不稳。人容易被熟悉感、假线索、先入为主和工具幻觉误导。

### 3. 瞄准镜期

真正的突破不是拥有更强工具，而是拥有更高的认知精度。

人开始追问：

- 我为什么会错？
- 我被什么骗了？
- 如何校准误差？

## 我想讨论的问题

- 工具升级之后，目标判断有没有同步升级？
- AI 是在帮助我推进问题，还是帮我逃避真正的问题？
- 为什么“看起来懂了”有时比“不懂”更危险？
- 学习中的错误，能否被整理成一套可复盘的误差地图？
- 在强工具时代，人的主体性应该放在哪里？

## 如何运行

这是一个纯静态网页，不需要安装依赖。

直接打开：

```text
index.html
```

或者启动一个静态服务器：

```bash
python3 -m http.server 8000
```

然后访问：

```text
http://localhost:8000
```
