# 从弹弓到瞄准镜 / From Slingshot to Scope

一个关于学习过程中，使用AI工具辅助备考产生了一点对认知误差与自我校准的小想法
AI Coding 思想表达交互实验

目前技术能力不足以做工具型开源项目，它更像展示一个小型认知模型：把一次真实学习复盘里的感受，转译成可以被拖动、观看和追问的代码原型。

这个仓库重点记录的是：一个想法如何被整理、压缩、结构化，并最终变成一个可交互的网页。

打开 `index.html` 即可体验。

## 项目缘起

这个项目来自一次真实的学习复盘。

我发现自己真正的提升不再是“知道更多”，
而是能开始判断自己可能会如何被误导或是被先入为主牵着走，
也会用工具制造一种“我好像懂了”的幻觉。

于是我把这种感觉转化为一个隐喻：

> 从“弹弓期”到“拿到枪但仍然模糊”，再到“拥有瞄准镜”的过程。

## 核心隐喻

### 1. 弹弓期

工具弱，目标也模糊。

人靠兴趣、冲动、零散尝试行动，不知道自己真正瞄准什么。

### 2. 枪但近视期

工具变强了。

AI工具、资料教程都在手里，但判断能力还不稳。
人容易被熟悉感、假线索、先入为主和工具幻觉误导。

### 3. 瞄准镜期

真正的突破不是拥有更强工具，而是拥有更高的认知精度。
开始追问：
- 我为什么会错？
- 我被什么骗了？
- 如何校准误差？

## 我想讨论的问题

- 工具升级之后，目标判断有没有同步升级？
- AI 是在帮助我推进问题，还是帮我逃避真正的问题？
- 为什么“看起来懂了”有时比“不懂”更危险？
- 学习中的错误，能否被整理成一套可复盘的误差地图？
- 在强工具时代，人的主体性应该放在哪里？
